# form_controller

A simple little handler for TextEditingControllers in forms.

## Getting Started

### 1. Depend on it

Add this to your package's pubspec.yaml file:

```
dependencies:
  form_controller: [latest version]
```

### 2. Install it

```
$ flutter pub get
```

### 3. Import it

```
import 'package:form_controller/form_controller.dart';
```
